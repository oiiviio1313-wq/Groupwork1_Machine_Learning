# Groupwork1_Machine_Learning
基于从前程无忧网爬取的**13,148**条职位数据，通过系统的数据清洗和特征 工程，采用机器学习方法对职位薪资进行预测分析。研究采用随机森林回归模型，深 入分析了影响薪资的关键因素。数据分析结果显示，工作经验、城市等级、职位类 型和公司规模是决定薪资水平的主要因素，为求职者职业规划和企业薪酬策略提供 了数据支持。

- spider.py: 抓包爬取[前程无忧](https://www.51job.com/)上**“大数据”**、**“算法”**、**“量化”**等13个关键词下所有职位信息。
- data_summary.py：对已爬取数据进行描述性统计
- data_cleaning.py, data_cleaning_salary.py：数据清洗，包括去重，处理缺失值，规范化城市名称、职位名称和薪资，并去除与“大数据”专业相关性较弱的职位数据
- average_salary.py, city_salary.py, JobType_Salary.py, ciyuntu.py: 根据清洗后的分职业数据和职业总数据绘制图表，包括：**不同职位平均薪资对比图**，**不同城市等级薪资分布对比图**，**六类职位平均薪资对比图**和**职位描述词云图**
- model.py：机器学习薪资预测模型，通过分析职位数据中的城市等级、岗位类型和经验学历等多维度特征，使用并评估**随机森林、XGBoost、SVM**等算法进行岗位**薪资分层预测**，选出**最佳模型**进行详细评估，输出相关图表。
